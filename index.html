Settings & API keys

All keys are optional and stored only in your browser (localStorage) — sent directly to the named provider's API, never anywhere else. Prefer not to use this dialog at all? Open the page source and fill in DEFAULT_KEYS near the top of the <script> instead, and every visitor to your deployment gets threat data with no setup.

Always-on (no key needed)
Repeated lookups of the same IP within this window are served from localStorage instead of re-querying every API. Set to 0 to disable caching.
AbuseIPDB
Abuse-confidence score & report counts. Note: some AbuseIPDB tiers block direct browser (CORS) requests — if so the panel will say so.
VirusTotal
Vendor detection counts for the IP. VirusTotal's API commonly blocks direct browser requests — the panel will tell you if that happens.
GreyNoise Community
Classifies whether an IP is internet background-noise (mass scanners) — malicious, benign, or unknown.
IPQualityScore
Fraud score plus VPN/proxy/bot indicators, cross-checked against the ipwho.is flags.
AlienVault OTX
Pulse count — how many threat-intel reports reference this IP.
